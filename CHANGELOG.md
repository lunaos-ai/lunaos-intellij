# Changelog

All notable changes to the LunaOS IntelliJ plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-30

### Added

- Agent browser tool window with search and category filtering
- One-click workflow execution with dynamic parameter forms
- Real-time log streaming panel with level filtering
- Run history panel showing status, duration, and outputs
- Editor context menu action to analyze selected code with AI agents
- Status bar widget displaying active run count
- Configurable API endpoint and key in Settings > Tools > LunaOS
- Tools menu group with Run Agent (Ctrl+Alt+L), Browse Agents, and View Recent Runs actions
- Notification support via balloon notifications for run completion and errors
- Custom agent cell renderer with category badges and descriptions
- Support for IntelliJ 2024.1 through 2026.1
- OkHttp-based API client for LunaOS platform communication
